Collection of string utility functions

## Install

Deno

```shell
deno add @utility/string
```

Node

```shell
npx jsr add @utility/string
```

Bun

```shell
bunx jsr add @utility/string
```

## Usage

```typescript
import { underscoreToTitleCase } from "@utility/string";

underscoreToTitleCase("hello_world");
// => Hello World
```
