# @utility/string

A String utility library

Deno

```shell
deno add @utility/string
```

Node

```shell
npx jsr add @utility/string
```

Bun

```shell
bunx jsr add @utility/string
```

Example

```typescript
import { underscoreToCamelCase } from "@utility/string";

underscoreToCamelCase("hello_world");
// => Hello World
```
